2 The Gaussian process regression model

Definition 6. Gaussian process (GP) is a collection of random variables { f ( x ) ; x ∈ X } , indexed by label x , where any finite collection of those variables has a multivariate normal distribution. It is fully specified by its mean and covariance functions. It is denoted as f ( ⋅ ) ∼ GP ( μ ( ⋅ ) , C ( ⋅ , ⋅ ) ) with mean μ ( x ) : = E ( f ( x ) ) , x ∈ X and covariance function C ( x , x ' ) : = Cov ( f ( x ) , f ( x ' ) ) , x , x ' ∈ X

Note 7. Essentially, GP is a distribution defined over functions.

Example 8. One way to simulate a GP realization f ( ⋅ ) ∼ GP ( μ ( ⋅ ) , C ( ⋅ , ⋅ ) ) , with μ ( x ) = 1 + 0.01 x , and C ( x , x ' ) = exp ( - 1 2 | x - x ' | 2 1.5 ) in the range x ∈ [ 0 , 10 ] is given in the following R code.

Note 9. Consider a function η : X → R with η ( x ) = ⟨ ψ ( x ) , w ⟩ where ψ ( x ) is a vector of known basis (feature) functions mapping from the input space X to the feature space F , and w ∈ R d is an unknown vector a priori following a normal distribution w ∼ N ( 0 , V ) , where the prior mean is set to zero denoting complete uncertainty about the sign of w ’s. Then the marginal η ( ⋅ ) follows a Normal distribution as a linear transformation of Normal variates with mean E ( η ( x ) ) = 0 and covariance Cov ( η ( x ) , η ( x ' ) ) = ψ ( x ) ⊤ V ψ ( x ' ) for any x , x ' ∈ X . 4 Examples of Covariance Functions


Note 27. The covariance function C : X × X → [ 0 , ∞ ) with C ( x , x ' ) describes how much two random variables x , x ' change together.

Note 28. Covariance function is a functional parameter of the GP prior (2.2). Different covarince functions represent different properties, hence they impose different prior info in the GP regression model; they are crusial parameters.

Note 29. Any positive definite Kernel as described in Section sec:Construction-of-kernels in Handout 7 Kernel methods can be used as a covarince function. Consequently kernel construction approached and theories introduced can be use for the covariance functions as well.

Definition 30. Stationary covariance function is called a covariance function C : X × X → [ 0 , ∞ ) whose image can be written as C ( x , x ' ) = C ( ∥ x - x ' ∥ ) namely, the dependence between any pair of input points x , x ' is a function of their distance and only.

Note 31. In 1 -D Gaussian process, one way to understand the characteristic length-scale of the process (if this exists) is in terms of the number of upcrossings of a level u . Consider stationary covariance function C ( x , x ' ) = C ( ∥ x - x ' ∥ ) . The expected number of upcrossings E ( N u ) of the level u on the unit interval by a zero-mean, stationary, is E ( N u ) = 1 2 π - C ' ' ( 0 ) C ' ( 0 ) exp ( - u 2 2 C ( 0 ) )

Note 32. Popular covariance functions are
Gaussian covariance function
given as C ( r ) = exp ( - 1 2 ϕ 2 r 2 )
  • It is infinitely differentiable, which means that the GP is very smooth.
  • The parameter ϕ is called lengthscale.
  • The number of upcrossing at level u is E ( N u ) = ( 2 ϕ 2 ) - 1 meaning that smaller ϕ represents more upcrossings, hence represents smaller scale dependences
Exponential Covariance Function
given as C ( r ) = exp ( - 1 ϕ | r | )
  • It is not differentiable at r = 0
Matern Class of Covariance Functions
given as C ν ( r ) = 2 1 - ν Γ ( ν ) ( 2 ν r ϕ ) ν B ν ( 2 ν r ϕ ) where B ν ( ⋅ ) is a modified Bessel functions (description of Bessel functions is out of the scope). Matern covariance function gives the Exponential one for ν = 1 / 2 , and the Gaussian one for ν → ∞ .

5 Practical maters

Note 36. One may consider some low degree polynomial form μ ( x | β ) = ∑ j = 0 p x j β j , however in this case μ ( ⋅ | β ) and C ( ⋅ , ⋅ | ϕ ) may compete as C ( ⋅ , ⋅ | ϕ ) can express such behaviors according to Note 9. For this reason, the usual specification of μ ( ⋅ | β ) in (2.2) is μ ( x | β ) = 0 implying a priori complete uncertainty about the sign of η ( x ) at each x .

Note 37. (Geostistical model) Thinking as a statistician, one may decompose (2.2) as η ( ⋅ ) = μ ( ⋅ | β ) + ξ ( ⋅ | ϕ ) where μ ( ⋅ | β ) is modeled as a low degree polynomial (e.g., 2nd degree) representing large scale dependences (see polynomial regression in Term 1), and ξ ( ⋅ | ϕ ) ∼ GP ( 0 , C ( ⋅ , ⋅ | ϕ ) ) representing lower scale dependence by using an appropriate kernel. Seeing the big picture, (2.1) can be re-stated as y i = μ ( x i | β ) + ξ ( x i | ϕ ) + ϵ i where therm ϵ i represents noise (no dependence or so short scale dependence that can be considered as noise in the model), ξ ( x i | ϕ ) represents low-scale dependence (about nearby inputs points), and μ ( x i | β ) represents large scale dependence (about very distant inputs points).

Definition 38. A d -dimensional random variable x ∈ R d is said to have a multivariate Normal (Gaussian) distribution, if for every d -dimensional fixed vector α ∈ R d , the random variable α ⊤ x has a univariate Normal (Gaussian) distribution.

Definition 39. We denote the d -dimensional Normal distribution with mean μ and covariance matrix Σ ≥ 0 as N d ( μ , Σ ) .

Notation 40. The d -dimensional standardized Normal distribution is N d ( 0 , I ) .

Proposition 41. Let random variable x ∼ N d ( μ , Σ ) , fixed vector c ∈ R q and fixed matrix A ∈ R q × R d . The random vector y = c + A x has distribution y ∼ N q ( c + A μ , A Σ A ⊤ ) .

Proposition 42.  Let a d -dimensional random vector x ∼ N (any ) ( μ , Σ ) .
  1. Let y = A x and z = B x , where A ∈ R q × d and B ∈ R k × d : The vectors y = A x and z = B x are independent if and only if A Σ B ⊤ = 0 .
  2. Let x = ( x 1 , ... , x d ) ⊤ : The x 1 , ... , x d are mutually independent if and only if the corresponding off diagonal parts of the Σ are zero.

Proposition 43. Any sub-vector of a vector with multivariate Normal distribution has a multivariate Normal distribution.

Proposition 44. [Marginalization & conditioning] Let x ∼ N d ( μ , Σ ) . Consider partition such that x = [ x 1 x 2 ] ; μ = [ μ 1 μ 2 ] ; Σ = [ Σ 1 Σ 21 ⊤ Σ 21 Σ 2 ] , where x 1 ∈ R d 1 , and x 2 ∈ R d 2 Then:
  1. For the marginal, it is x 1 ∼ N d 1 ( μ 1 , Σ 1 ) .
  2. For the conditional, if Σ 1 > 0 , it is x 2 | x 1 ∼ N d 2 ( μ 2 | 1 , Σ 2 | 1 ) where μ 2 | 1 = μ 2 + Σ 21 Σ 1 - 1 ( x 1 - μ 1 ) and Σ 2 | 1 = Σ 2 - Σ 21 Σ 1 - 1 Σ 21 ⊤ (A.1)

Proposition 45. The density function of the d -dimensional Normal distribution with mean μ and covariance matrix Σ , when Σ is symmetric positive definite matrix ( Σ > 0 ), exists and it is equal to f ( x ) = ( 2 π ) - d 2 det ( Σ ) - 1 2 exp ( - 1 2 ( x - μ ) ⊤ Σ - 1 ( x - μ ) ) (A.2)
