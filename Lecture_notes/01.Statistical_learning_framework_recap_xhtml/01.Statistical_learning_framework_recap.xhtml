1 General Introductions and loose definitions

Pattern recognition is the automated discovery of patterns and regularities in data z ∈ Z . Machine learning (ML) are statistical procedures for building and understanding probabilistic methods that ’learn’. ML algorithms A build a (probabilistic/deterministic) model able to make predictions or decisions with minimum human interference and can be used for pattern recognition. Learning (or training, estimation, fitting) is called the procedure where the ML model is tuned. Training data (or observations, sample data set, examples) is a set of observables { z i ∈ Z } used to tune the parameters of the ML model. By Z we denote the examples (or observables) domain. Test set is a set of available examples/observables { z ' i } (different than the training data) used to verify the performance of the ML model for a given a measure of success. Measure of success (or performance) is a quantity that indicates how bad the corresponding ML model or Algorithm performs (eg quantifies the failure/error), and can also be used for comparisons among different ML models; eg, Risk function or Empirical Risk Function. Two main problems in ML are the supervised learning (we will focus on this here) and the unsupervised learning.
Supervised learning problems involve applications where the training data z ∈ Z comprise examples of the input vectors x ∈ X along with their corresponding target vectors y ∈ Y ; i.e. Z = X × Y . By X we denote the inputs (or instances) domain, and by Y we denote the target domain. Classiﬁcation problems are those which aim to assign each input vector x to one of a ﬁnite number of discrete categories of y . Regression problems are those where the output y consists of one or more continuous variables. All in all, the learner wishes to discover an unknown pattern (i.e. functional relationship) between components x ∈ X that serves as inputs and components y ∈ Y that act as outputs; i.e. . Hence, X is the input domain, and Y is the output (or target) domain. The goal of learning is to discover a function which predicts (or help us make decisions about) y ∈ Y from x ∈ X .
Unsupervised learning problems involve applications where the training data z ∈ Z consist of a set of input vectors x ∈ X without any corresponding target values ; i.e. Z = X . In clustering the goal is to discover groups of similar examples within the data of it is to discover groups of similar examples within the data.

2 (loose) Notation & definitions in learning

Definition 1. The learner’s output is a function, h : X → Y which predicts y ∈ Y from x ∈ X . It is also called Hypothesis, prediction rule, predictor, or classifier.

Notation 2. We often denote the set of hypothesis as H ; i.e. h ∈ H .

Example 3. (Linear Regression)

Consider the regression problem where the goal is to learn the mapping x → y where x ∈ X ⊆ R d and y ∈ Y ⊆ R . A hypothesis is a linear function h : X → Y (that learner wishes to learn) with h ( x ) = ⟨ w , x ⟩ approximating the mapping x → y . The hypothesis set H = { x → ⟨ w , x ⟩ : w ∈ R d } .

Example 4. (Binary Classification) Consider the classification problem where the goal is to learn the mapping x → y where x ∈ X ⊆ R d and y ∈ Y { - 1 , + 1 } . A hypothesis can be a function h : X → Y with h ( x ) = sign ( ⟨ w , x ⟩ ) approximating the mapping x → y . The hypothesis set H = { x → sign ( ⟨ w , x ⟩ ) : w ∈ R d } .

Definition 5. Training data set S of size m is any finite sequence of pairs ( z i = ( x i , y i ) ; i = 1 , ... , m ) in X × Y ; i.e. S = { ( x i , y i ) ; i = 1 , ... , m } . This is the information that the learner has assess.

Definition 6. Data generation model g ( ⋅ ) is the probability distribution over Z = X × Y , unknown to the learner that has generated the data. E.g. z ∼ g .

Definition 7. We denote as A ( S ) the hypothesis (outcome) that a learning algorithm A returns given training sample S .

Definition 8. (Loss function) Given any set of hypothesis H and some domain Z = X × Y , a loss function ℓ ( ⋅ ) is any function ℓ : H × Z → R + . Loss function ℓ ( h , z ) for h ∈ H and z ∈ Z is specified according to the purpose the machine learning algorithm. It reflects how the “error” is quantified for a given hypothesis h and a given example z . The rule is “the greater the error the greater the value of the loss”.

Example 9. (Cont. Example 3) In regression problems Z = X × Y and Y ⊂ R is uncountable, a potential loss function is ℓ sq ( h , ( x , y ) ) = ( h ( x ) - y ) 2

Example 10. (Cont. Example 4) In binary classification problems with hypothesis h : X → Y where Y = { 0 , 1 } is discrete, a loss function can be ℓ 0 - 1 ( h , ( x , y ) ) = 1 ( h ( x ) ≠ y ) ,

Definition 11. (Risk function) The risk function R g ( h ) of h is the expected loss of the hypothesis h ∈ H , w.r.t. the data generation model (which is a probability distribution) g over domain Z ; i.e. R g ( h ) = E z ∼ g ( ℓ ( h , z ) ) (2.1)
Remark 12. In learning, an ideal way to obtain an optimal predictor h * is to compute the minimizer of the risk; i.e. h * = arg min ∀ h ( R g ( h ) ) (2.2)

Example 13. (Cont. Ex. 9) The risk function is R g ( h ) = E z ∼ g ( h ( x ) - y ) 2 , and it measures the quality of the hypothesis function h : X → Y , (or equiv. the validity of the class of hypotheses H ) against the data generating model g , as the expected square difference between the predicted values form h and the true target values y at every x .

Note 14. Computing the risk minimizer may be practically challenging due to the integration w.r.t. the unknown data generation model g involved in the expectation (2.1). Sub-optimally, one may use the Empirical risk function instead of the Risk function in (2.2).

Definition 15. (Empirical risk function) The Empirical Risk Function (ERF) R ˆ S ( h ) of h is the expectation of loss of h over a given sample S = ( z 1 , ... , z m ) ∈ Z m ; i.e. R ˆ S ( h ) = 1 m ∑ i = 1 m ℓ ( h , z i ) .

Remark 16. Given Empirical Risk Function (ERF) R ˆ S ( h ) of h the optimal predictor h * is the minimizer of the ERF; i.e. h * = arg min ∀ h ( R ˆ S ( h ) ) (2.3)

Example 17. (Cont. Example 13) Given given sample S = { ( x i , y i ) ; i = 1 , ... , m } the empirical risk function is R ˆ S ( h ) = 1 m ∑ i = 1 m ( h ( x i ) - y i ) 2 .

Example 18. (Cont. Example 10) Given given sample S = { ( x i , y i ) ; i = 1 , ... , m } the empirical risk function is R ˆ S ( h ) = 1 m ∑ i = 1 m 1 ( h ( x i ) ≠ y i ) .

Definition 22. A learning problem with hypothesis class H , examples domain Z , and loss function ℓ may be denoted with a triplet ( H , Z , ℓ ) .

Example 23. The standard multiple linear regression problem with regressors x ∈ X ⊆ R d and response y ∈ Y ⊆ R , is a learning problem with examples domain Z = X × Y = R d + 1 , hypothesis class H = { x → ⟨ w , x ⟩ : w ∈ R d } , and loss function ℓ sq ( h , ( x , y ) ) = ( h ( x ) - y ) 2 .
