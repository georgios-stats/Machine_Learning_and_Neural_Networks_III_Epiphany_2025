1 Motivations


Note 1. We introduce concepts of convexity and smoothness that facilitate the analysis and understanding of the learning problems and their solutions that we will discuss (eg stochastic gradient descent, SVM) later on. Also learning problems with such characteristics can be learned more efficiently.

Note 2. Some of the ML problems discussed in the course (eg, Artificial neural networks, Gaussian process regression) are non-convex. To overcome this problem, we will introduce the concept of surrogate loss function that allows a non-convex problem to be handled with the tools introduced int he convex setting.

4 Lipschitzness

Definition 19. Let C ∈ R d . Function f : R d → R k is ρ -Lipschitz over C if for every w 1 , w 2 ∈ C we have that ∥ f ( w 1 ) - f ( w 2 ) ∥ ≤ ρ ∥ w 1 - w 2 ∥ . Lipschitz condition (4.1)

Conclusion 20. That means: a Lipschitz function f ( x ) cannot change too drastically wrt x .
Example 21. Consider the function f : R → R + with f ( x ) = x 2 .

6 Convex learning problems

Definition 30. Convex learning problem is a learning problem ( H , Z , ℓ ) that the hypothesis class H is a convex set, and the loss function ℓ is a convex function for each example z ∈ Z .

Example 31. Consider the regression problem with predictor rule h ( x ) = ⟨ w , x ⟩ , loss function ℓ ( w , ( x , y ) ) = ( ⟨ w , x ⟩ - y ) 2 , feature x ∈ R d , and target y ∈ R . This imposes a convex learning problem due to Examples 6 and 13.

Definition 32. Convex-Lipschitz-Bounded Learning Problem ( H , Z , ℓ ) with parameters ρ , and B , is called the learning problem whose the hypothesis class H is a convex set, for all w ∈ H it is ∥ w ∥ ≤ B , and the loss function ℓ ( ⋅ , z ) is convex and ρ -Lipschitz function for all z ∈ Z .

Example 33. Consider the regression problem with predictor rule h ( x ) = ⟨ w , x ⟩ , loss function ℓ ( w , ( x , y ) ) = ( ⟨ w , x ⟩ - y ) 2 , feature x ∈ R d , and target y ∈ R . This imposes a Convex-Lipschitz-Bounded Learning Problem if H = { w ∈ R d : ∥ w ∥ ≤ B } due to Examples 13, and 21(2.).

Definition 34. Convex-Smooth-Bounded Learning Problem ( H , Z , ℓ ) with parameters β , and B , is called the learning problem whose the hypothesis class H is a convex set, for all w ∈ H it is ∥ w ∥ ≤ B , and the loss function ℓ ( ⋅ , z ) is convex, nonnegative, and β -smooth function for all z ∈ Z .

Example 35. Consider the regression problem with predictor rule h ( x ) = ⟨ w , x ⟩ , loss function ℓ ( w , ( x , y ) ) = ( ⟨ w , x ⟩ - y ) 2 , feature x ∈ R d , and target y ∈ R . This imposes a Convex-Smooth-Bounded Learning Problem if H = { w ∈ R d : ∥ w ∥ ≤ B } due to Examples 13, and 29.
Proposition 36. If ℓ is a convex loss function and the class H is convex, then the ERM H problem, of minimizing the empirical risk R ˆ S ( w ) over H , is a convex optimization problem (that is, a problem of minimizing a convex function over a convex set).
Proof. The ERM H problem is w * = arg min w ∈ H { R ˆ S ( w ) } given a sample S = { z 1 , ... , z m } for R ˆ S ( w ) = 1 m ∑ i = 1 m ℓ ( w , z i ) . R ˆ S ( w ) is a convex function from Proposition (13). Hence ERM rule is a problem of minimizing a convex function subject to the constraint that the solution should be in a convex set.

7 Non-convex learning problems (surrogate treatment)

Remark 39. A learning problem may involve non-convex loss function ℓ ( w , z ) which implies a non-convex risk function R g ( w ) . However, our learning algorithm will be analyzed in the convex setting. A suitable treatment to overcome this difficulty would be to upper bound the non-convex loss function ℓ ( w , z ) by a convex surrogate loss function ℓ ˜ ( w , z ) for all w , and use ℓ ˜ ( w , z ) instead of ℓ ( w , z ) .
Example 40. Consider the binary classification problem with inputs x ∈ X , outputs y ∈ { - 1 , + 1 } ; we need to learn w ∈ H from hypothesis class H ⊂ R d with respect to the loss ℓ ( w , ( x , y ) ) = 1 ( y ⟨ w , x ⟩ ≤ 0 ) with y ∈ R , and x ∈ R d . Here ℓ ( ⋅ ) is non-convex. A convex surrogate loss function can be ℓ ˜ ( w , ( x , y ) ) = max ( 0 , 1 - y ⟨ w , x ⟩ ) which is convex (Example 13) wrt w . Note that:
  • ℓ ˜ ( w , ( x , y ) ) is convex wrt w ; because max ( ⋅ ) is convex
  • ℓ ( w , ( x , y ) ) ≤ ℓ ˜ ( w , ( x , y ) ) for all w ∈ H
Then we can compute w ˜ * = arg min ∀ x ( R ˜ g ( w ) ) = arg min ∀ x ( E ( x , y ) ∼ g ( max ( 0 , 1 - y ⟨ w , x ⟩ ) ) ) instead of w * = arg min ∀ x ( R g ( w ) ) = arg min ∀ x ( E ( x , y ) ∼ g ( 1 ( y ⟨ w , x ⟩ ≤ 0 ) ) ) Of course by using the surrogate loss instead of the actual one, we introduce some approximation error in the produced output w ˜ * ≠ w * .

Remark 41. (Intuitions...) Using a convex surrogate loss function instead the convex one, facilitates computations but introduces extra error to the solution. If R g ( ⋅ ) is the risk under the non-convex loss, R ˜ g ( ⋅ ) is the risk under the convex surrogate loss, and w ˜ alg is the output of the learning algorithm under R ˜ g ( ⋅ ) then we have the upper bound R g ( w ˜ alg ) ≤ min w ∈ H ( R g ( w ) ) ⏟ I + ( min w ∈ H ( R ˜ g ( w ) ) - min w ∈ H ( R g ( w ) ) ) ⏟ II + ϵ ⏟ III where term I is the approximation error measuring how well the hypothesis class performs on the generating model, term II is the optimization error due to the use of surrogate loss instead of the actual non-convex one, and term III is the estimation error due to the use of a training set and not the whole generation model.
