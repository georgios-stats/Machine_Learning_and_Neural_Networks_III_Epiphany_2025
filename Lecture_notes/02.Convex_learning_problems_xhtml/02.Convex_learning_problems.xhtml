1 Motivations


Note 1. We introduce learning problems associated with convexity, Lipschitzness and smoothness to be able to analyze and understand the machine learning (ML) tools we will discuss later on (eg stochastic gradient descent, SVM). We discuss ways allowing to address non-convex ML problems (eg, Artificial neural networks, Gaussian process regression) in the convex setting.

2 Convexity

Note 2. Convexity is a central concept in learning, e.g. least squares, as it often considers a Euclidean distance as a Risk function to be minimized.

Definition 3. A set C is convex if for any u , v ∈ C and for any α ∈ [ 0 , 1 ] we have that α u + ( 1 − α ) v ∈ C .

Definition 5. Let C be a convex set. A function f : C → R is convex function if for any u , v ∈ C and for any α ∈ [ 0 , 1 ] f ( α u + ( 1 − α ) v ) ≤ α f ( u ) + ( 1 − α ) f ( v )

3 Strong convexity

Note 13. Strong convexity is a central concept in Ridge regularization, as it makes a convex loss function strongly convex by adding a shrinkage term.

Definition 14. (Strongly convex functions) A function f is λ -strongly convex function is for all w , u , and α ∈ ( 0 , 1 ) we have f ( α w + ( 1 - α ) u ) ≤ a f ( w ) + ( 1 - α ) f ( u ) - λ 2 α ( 1 - α ) ∥ w - u ∥ 2 (3.1)

Note 15. The following can be checked from the definition by substitution
  1. The function f ( w ) = λ ∥ w ∥ 2 is 2 λ -strongly convex
  2. If f is λ -strongly convex and g is convex then f + g is λ -strongly convex


4 Lipschitzness

Definition 16. Let C ⊆ R d . Function f : C → R k is ρ -Lipschitz over C if for every w 1 , w 2 ∈ C we have that ∥ f ( w 1 ) - f ( w 2 ) ∥ ≤ ρ ∥ w 1 - w 2 ∥ . Lipschitz condition (4.1)

Note 17. That means: a Lipschitz function f ( x ) cannot change too drastically wrt x .
Example 18. Consider the function f : R → R + with f ( x ) = x 2 .
  1. f is not a ρ -Lipschitz in R .
  2. f is a ρ -Lipschitz in C = { x ∈ R : | x | < ρ / 2 } .
Solution.  
  1. For x 1 = 0 and x 2 = 1 + ρ , it is | f ( x 2 ) - f ( x 1 ) | = ( 1 + ρ ) 2 > ρ ( 1 + ρ ) = ρ | x 2 - x 1 |
  2. It is | f ( x 2 ) - f ( x 1 ) | = | x 2 2 - x 1 2 | = | ( x 2 + x 1 ) ( x 2 - x 1 ) | ≤ 2 ρ / 2 ( x 2 - x 1 ) = ρ | x 2 - x 1 |

Note 19. Let functions g 1 be ρ 1 -Lipschitz and g 2 be ρ 2 -Lipschitz. Then f with f ( x ) = g 1 ( g 2 ( x ) ) is ρ 1 ρ 2 -Lipschitz.

6 Convex learning problems

Definition 27. Convex learning problem is a learning problem ( H , Z , ℓ ) that the hypothesis class H is a convex set, and the loss function ℓ ( ⋅ , z ) is a convex function for each example z ∈ Z .

Example 28. Consider the regression problem with predictor rule h ( x ) = ⟨ w , x ⟩ , loss function ℓ ( w , ( x , y ) ) = ( ⟨ w , x ⟩ - y ) 2 , feature x ∈ R d , and target y ∈ R . This imposes a convex learning problem due to Examples 4 and 11.

Definition 29. Convex-Lipschitz-Bounded Learning Problem ( H , Z , ℓ ) with parameters ρ , and B , is called the learning problem whose the hypothesis class H is a convex set, for all w ∈ H it is ∥ w ∥ ≤ B , and the loss function ℓ ( ⋅ , z ) is convex and ρ -Lipschitz function for all z ∈ Z .

Example 30. Consider the regression problem with predictor rule h ( x ) = ⟨ w , x ⟩ , loss function ℓ ( w , ( x , y ) ) = ( ⟨ w , x ⟩ - y ) 2 , feature x ∈ R d , and target y ∈ R . This imposes a Convex-Lipschitz-Bounded Learning Problem if H = { w ∈ R d : ∥ w ∥ ≤ B } due to Examples 11, and 18(2.).

Definition 31. Convex-Smooth-Bounded Learning Problem ( H , Z , ℓ ) with parameters β , and B , is called the learning problem whose the hypothesis class H is a convex set, for all w ∈ H it is ∥ w ∥ ≤ B , and the loss function ℓ ( ⋅ , z ) is convex, nonnegative, and β -smooth function for all z ∈ Z .

Example 32. Consider the regression problem with predictor rule h ( x ) = ⟨ w , x ⟩ , loss function ℓ ( w , ( x , y ) ) = ( ⟨ w , x ⟩ - y ) 2 , feature x ∈ R d , and target y ∈ R . This imposes a Convex-Smooth-Bounded Learning Problem if H = { w ∈ R d : ∥ w ∥ ≤ B } due to Examples 11, and 26.

Note 33. (Empirical risk minimization learning problem) If ℓ is a convex loss function and the class H is convex, then the ERM H problem, of minimizing the empirical risk R ˆ S ( w ) over H , is a convex optimization problem
Proof. The Empirical risk minimization (ERM) problem ( H , Z , ℓ ) is w * = arg min w ∈ H { R ˆ S ( w ) } given a sample S = { z 1 , ... , z m } for R ˆ S ( w ) = 1 m ∑ i = 1 m ℓ ( w , z i ) . R ˆ S ( w ) is a convex function from Note (11). Hence ERM rule is a problem of minimizing a convex function subject to the constraint that the solution should be in a convex set.
Example 34. Multiple linear regression with predictor rule h ( x ) = ⟨ w , x ⟩ , loss function ℓ ( w , ( x , y ) ) = ( ⟨ w , x ⟩ - y ) 2 , feature x ∈ R d , and target y ∈ R where w * = arg min w E ( ⟨ w , x ⟩ - y ) 2 or w * * = arg min w 1 m ∑ i = 1 m ( ⟨ w , x i ⟩ - y i ) 2 is a convex learning problem –from Note 33.

Note 35. (Ridge / ℓ 2 -RLM problem) If ℓ is a convex loss function w.r.t. w , the class H is convex, and J ( ⋅ ; λ ) = λ ∥ ⋅ ∥ 2 2 with λ > 0 then R ˆ S ( w ) + λ ∥ w ∥ 2 2 is a 2 λ -strongly convex function, and hence the ℓ 2 -RLM problem w * = arg min w ∈ H { R ˆ S ( w ) + λ ∥ w ∥ 2 2 } is a strongly convex optimization problem (i.e. the learning rule is the minimizer of a strongly convex function over a convex set).
Proof. R ˆ S ( ⋅ ) is a convex function from Note 33, λ ∥ ⋅ ∥ 2 2 is 2 λ -strongly convex, hence R ˆ S ( w ) + λ ∥ w ∥ 2 2 is a 2 λ -strongly convex function. Hence the above Ridge RLM problem is a strongly convex optimization problem.

7 Non-convex learning problems (surrogate treatment)

Note 36. A learning problem may involve non-convex loss function ℓ ( w , z ) w.r.t. w which implies a non-convex risk function R g ( w ) . A suitable treatment to analyze the associated learning tools within the convex setting would be to upper bound the non-convex loss function ℓ ( w , z ) by a convex surrogate loss function ℓ ˜ ( w , z ) for all w , and use ℓ ˜ ( w , z ) instead of ℓ ( w , z ) .
Example 37. Consider the binary classification problem with inputs x ∈ X , outputs y ∈ { - 1 , + 1 } ; we need to learn w ∈ H from hypothesis class H ⊂ R d with respect to the loss ℓ ( w , ( x , y ) ) = 1 ( y ⟨ w , x ⟩ ≤ 0 ) with y ∈ R , and x ∈ R d . Here ℓ ( ⋅ ) is non-convex. A convex surrogate loss function can be ℓ ˜ ( w , ( x , y ) ) = max ( 0 , 1 - y ⟨ w , x ⟩ ) which is convex (Example 11) wrt w . Note that:
  • ℓ ˜ ( w , ( x , y ) ) is convex wrt w ; because max ( ⋅ ) is convex
  • ℓ ( w , ( x , y ) ) ≤ ℓ ˜ ( w , ( x , y ) ) for all w ∈ H
Then we can compute w ˜ * = arg min ∀ x ( R ˜ g ( w ) ) = arg min ∀ x ( E ( x , y ) ∼ g ( max ( 0 , 1 - y ⟨ w , x ⟩ ) ) ) instead of w * = arg min ∀ x ( R g ( w ) ) = arg min ∀ x ( E ( x , y ) ∼ g ( 1 ( y ⟨ w , x ⟩ ≤ 0 ) ) ) Of course by using the surrogate loss instead of the actual one, we introduce some approximation error in the produced output w ˜ * ≠ w * .

Note 38. (Intuitions...) Using a convex surrogate loss function instead the convex one, facilitates computations but introduces extra error to the solution. If R g ( ⋅ ) is the risk under the non-convex loss, R ˜ g ( ⋅ ) is the risk under the convex surrogate loss, and w ˜ alg is the output of the learning algorithm under R ˜ g ( ⋅ ) then we have the upper bound R g ( w ˜ alg ) ≤ min w ∈ H ( R g ( w ) ) ⏟ I + ( min w ∈ H ( R ˜ g ( w ) ) - min w ∈ H ( R g ( w ) ) ) ⏟ II + ϵ ⏟ III where term I is the approximation error measuring how well the hypothesis class performs on the generating model, term II is the optimization error due to the use of surrogate loss instead of the actual non-convex one, and term III is the estimation error due to the use of a training set and not the whole generation model.
