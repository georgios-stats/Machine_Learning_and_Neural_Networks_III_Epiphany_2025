2 Improving expressive power via embeddings in feature spaces

Note 5. To make the class of hypotheses more expressive with purpose to improve accuracy, we can first map the original instance space x ∈ X into another feature space F (possibly of a higher dimension) via an embedding ψ : X → F and then learn a hypothesis in that space.

Note 6. Consider a given learning task that involves a hypothesis class where the predictive rule h ∈ H is defined over domain set X and is to be trained against data set S = { z i = ( x i , y i ) } i = 1 m drawn from data generating process G ( ⋅ ) . The basic paradigm for improving expressive power of H via embedding involves:
  1. Choose an embedding ψ : X → F with ψ ( x ) : = ( ψ 1 ( x ) , ... , ψ d ( x ) ) ⊤ for some feature space F .
  2. Create the image sequence S ψ = { z i ψ = ( ψ ( x i ) , y i ) } i = 1 m from the original training set S .
  3. Train a linear predictor h against S ψ .
  4. Predict the label or the output of a new point x new by h ψ ( x new ) : = h ○ ψ ( x new ) = h ( ψ ( x new ) )

Note 7. The introduction of embedding ψ : X → F induces
  1. a probability distribution G ψ over domain F × Y with G ψ ( A ) = G ( ψ - 1 ( A ) ) for every set A ⊆ F × Y given a data generating process G ( ⋅ ) .
  2. predictive rule h ψ ( ⋅ ) : = h ○ ψ ( ⋅ ) = h ( ψ ( ⋅ ) )
  3. risk function R G ψ ( h ) : = R G ( h ○ ψ ) , as R G ( h ○ ψ ) = ∫ ℓ ( h ○ ψ , z = ( x , y ) ) d G ( z ) = ∫ ℓ ( h , z ψ ) d G ψ ( x , y ) = R G ψ ( h )



Note 8. For a specific learning task, the success of the learning paradigm in Note 6 depends on choosing an embedding ψ that provides a suitable deformation for the image of the data generating process (or the training dataset) to be as close as possible to what could be accurately addressed by the specific learning task. Eg, in SVM, ψ will make the image of the data distribution (close to being) linearly separable in the feature space F , thus making the resulting learning algorithm a good learner for a given task (Figure 2.1). This requires prior knowledge of the problem (In Section 4, we see popular recipes for that).

Note 9. As feature mapping ψ any function that maps the original instances X into some Hilbert space F can be used.


Definition 10. A Hilbert space is a vector space, with an inner product, which is also complete.
Lemma 11. If X is a linear subspace of a Hilbert space, then every x ∈ X can be written as x = u + v where u ∈ X and ⟨ u , v ⟩ = 0 for all v ∈ X .

Note 12. Feature space F is a Hilbert space preferably due to Lemma 11 that enables the Kernel trick via the representation Theorem 19. Eg, a Euclidean space such as R d for some d . That includes infinite dimensional spaces.

Note 13. Using a ψ ( x ) = ( ψ 1 ( x ) , ψ 2 ( x ) ... ) ⊤ that is high dimensional ( d is too large) may improve accuracy (expressiveness) of the learner (e.g. recall in polynomial regression increasing the polynomial degree). However this increases the computational effort/cost required to perform calculations to minimize the associated risk function in the high dimensional space, as well as we need more data. This is addressed via the Kernel trick.

3 The Kernel trick

Note 15. We discuss a duality in the learning problem 16 that facilitates the implementation of the extension to a possibly high dimensional feature space (hence improving the expressiveness/accuracy) by using kernel functions (hence reducing dimensionality, computational cost, and required data size).

Problem 16. (Learning problem) Consider a prediction rule h : X → Y with h ( x ) = ⟨ w , ψ ( x ) ⟩ which is trained against a training sample { z i = ( x i , y i ) } i = 1 m with the following general optimization problem minimize w ( f ( ⟨ w , ψ ( x 1 ) ⟩ , ... , ⟨ w , ψ ( x m ) ⟩ ) + R ( ∥ w ∥ ) ) , (3.1) where f : R m → R is an arbitrary function and R : R + → R is a monotonically non-decreasing function.

Definition 18. Kernel function K is defined as K : X × X → C with K ( x , x ' ) = ⟨ ψ ( x ) , ψ ( x ' ) ⟩ given an embedding ψ ( x ) of some domain space X into some Hilbert space F . Kernel functions describe inner products in the feature space F .

Theorem 19. (Representation theorem) Assume mapping ψ : X → F where F is a Hilbert space. There exists a vector α ∈ R m such that w = ∑ i = 1 m α i ψ ( x i ) is the optimal solution of (3.1) in Problem 16.
Proof. Let w * be the optimal solution of (3.1). Because w * is element of Hilbert space, it can be written as w * = ∑ i = 1 m α i ψ ( x i ) + u where ⟨ u , ψ ( x i ) ⟩ = 0 for all i = 1 , ... , m . Set w : = w * - u .
Because ∥ w * ∥ 2 = ∥ w ∥ 2 + ∥ u ∥ 2 it is ∥ w ∥ ≤ ∥ w * ∥ implying that R ( ∥ w ∥ ) ≤ R ( ∥ w * ∥ ) . Because ⟨ w , ψ ( x i ) ⟩ = ⟨ w * - u , ψ ( x i ) ⟩ = ⟨ w * , ψ ( x i ) ⟩ for all i = 1 , ... , m , it is f ( ⟨ w , ψ ( x 1 ) ⟩ , ... , ⟨ w , ψ ( x m ) ⟩ ) = f ( ⟨ w * , ψ ( x 1 ) ⟩ , ... , ⟨ w * , ψ ( x m ) ⟩ ) Then the objective function of (3.1) at w is less than or equal to that of the minimizer w * which implies that w = ∑ i = 1 m α i ψ ( x i ) is an optimal solution.

4 Construction of kernels


Note 25. The kernel formulated as an inner product in a feature space allows us to build interesting extensions of many well-known algorithms by making use of the kernel trick and without the need to have direct access to the feature space (E.g. Example 22).

Note 26. Specifying a kernel function K is a way to express prior knowledge without the need to have direct access to the feature space. This is consequence of the Representation theorem 19 that kernel is the inner product of feature mappings ψ which sufficiently replaces them in the learning problem, and the fact that ψ is a way to express and utilize prior knowledge about the problem at hand.

Note 27. Theorem 30 provides sufficient and necessary conditions to check whether the specified function K : X × X → R is indeed a kernel function; i.e. if K can be written as inner product K ( x , x ' ) = ⟨ ψ ( x ) , ψ ( x ' ) ⟩ of feature functions ψ ( x ) .

Definition 28. Gram matrix is called the m × m matrix G s.t. [ G ] i , j = K ( x i , x j ) .

Definition 29. A symmetric function K : X × X → R is positive semi-definite if its Gram matrix G , [ G ] i , j = K ( x i , x j ) , is a positive semi-definite matrix.

Theorem 30. A symmetric function K : X × X → R implements an inner product in some Hilbert space is a valid kernel function if and only if it is positive semi-definite i.e. its Gram matrix G , [ G ] i , j = K ( x i , x j ) , is a positive semi-definite matrix.
Proof. Assume K is a valid kernel function (i.e. it implements an inner product in some Hilbert space) K ( x , x ' ) = ⟨ ψ ( x ) , ψ ( x ' ) ⟩ ; let’s consider ψ : X → R d for simplicity. Let G be its Gram matrix with G = Ψ ⊤ Ψ and ψ ( x i ) is the i -th column of Ψ . For any ξ ∈ R d - { 0 } ξ ⊤ G ξ = ∑ i ∑ j ξ i K ( x i , x j ) ξ j = ∑ i ∑ j ξ i ⟨ ψ ( x i ) , ψ ( x j ) ⟩ ξ j = ∑ i ∑ j ⟨ ξ i ψ ( x i ) , ψ ( x j ) ξ j ⟩ = ⟨ ∑ i ξ i ψ ( x i ) , ∑ j ψ ( x j ) ξ j ⟩ = ∥ ∑ i ξ i ψ ( x i ) ∥ 2 2 ≥ 0 Assume the symmetric function K : X × X → R is positive semi-definite. Let R f = { f : X → R } . For x ∈ X let function ψ over R f with ψ ( x ) = K ( ⋅ , x ) . This allows to define a vector space consisting of all the linear combinations of elements of the form K ( ⋅ , x ) , having an inner product ⟨ ∑ i α i K ( ⋅ , x i ) , ∑ j β j K ( ⋅ , x j ) ⟩ = ∑ i ∑ i α i β j ⟨ K ( ⋅ , x i ) , K ( ⋅ , x j ) ⟩ ⏟ = K ( x i , x j ) . This is satisfies all the properties of inner product, s.t. it is symmetric, linearity, positive definite as K ( x , x ' ) ≥ 0 . Then there is some feature vector ψ such that K ( x , x ' ) = ⟨ ψ ( x ) , ψ ( x ' ) ⟩ .
