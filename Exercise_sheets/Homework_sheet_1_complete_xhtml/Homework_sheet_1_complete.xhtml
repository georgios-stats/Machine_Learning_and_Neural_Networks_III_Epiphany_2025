Exercise 1.
   Let f : R d → R be a convex and β -smooth function.
  1. Show that for v , w ∈ R d f ( v ) - f ( w ) ∈ ( ⟨ ∇ f ( w ) , v - w ⟩ , ⟨ ∇ f ( w ) , v - w ⟩ + β 2 ∥ v - w ∥ 2 )
  2. Show that for v , w ∈ R d such that v = w - 1 β ∇ f ( w ) , it is 1 2 β ∥ ∇ f ( w ) ∥ 2 ≤ f ( w ) - f ( v )
  3. Additionally assume that f ( x ) > 0 for all x ∈ R d . Show that for w ∈ R d , ∥ ∇ f ( w ) ∥ ≤ 2 β f ( w )
Solution.    
  1. If f : R d → R is β -smooth then it is f ( v ) ≤ f ( w ) + ⟨ ∇ f ( w ) , v - w ⟩ + β 2 ∥ v - w ∥ 2 f ( v ) - f ( w ) ≤ ⟨ ∇ f ( w ) , v - w ⟩ + β 2 ∥ v - w ∥ 2 If it is convex then it is f ( v ) ≥ f ( w ) + ⟨ ∇ f ( w ) , v - w ⟩ f ( v ) - f ( w ) ≥ ⟨ ∇ f ( w ) , v - w ⟩ Together these conditions imply upper and lower bounds f ( v ) - f ( w ) ∈ ( ⟨ ∇ f ( w ) , v - w ⟩ , ⟨ ∇ f ( w ) , v - w ⟩ + β 2 ∥ v - w ∥ 2 )
  2. For v , w ∈ R d such that v = w - 1 β ∇ f ( w ) , it is f ( v ) ≤ f ( w ) + ⟨ ∇ f ( w ) , v - w ⟩ + β 2 ∥ v - w ∥ 2 2 (due to smoothness ) ⟺ f ( w ) - f ( v ) ≤ f ( w ) - f ( v ) ⟺ ⟨ ∇ f ( w ) , v - w ⟩ + β 2 ∥ v - w ∥ 2 2 ≤ f ( w ) - f ( v ) ⟺ ⟨ ∇ f ( w ) , 1 β ∇ f ( w ) ⟩ + β 2 ∥ 1 β ∇ f ( w ) ∥ 2 2 ≤ f ( w ) - f ( v ) ⟺ 1 2 β ∥ ∇ f ( w ) ∥ 2 ≤ f ( w ) - f ( v ) ∥ ∇ f ( w ) ∥ 2 ≤ 2 β ( f ( w ) - f ( v ) ) as f ( ⋅ ) ≥ 0 ∥ ∇ f ( w ) ∥ 2 ≤ 2 β f ( w )
  3. From part 2, this is obvious because f ( x ) > 0 for all x ∈ R d , as ∥ ∇ f ( w ) ∥ 2 ≤ 2 β f ( w ) ↔ ∥ ∇ f ( w ) ∥ ≤ 2 β f ( w )
